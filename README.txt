# 🗣️ Text-to-Speech JavaFX App

## 📌 Overview
This **Text-to-Speech (TTS) application** is built with **JavaFX** and uses **FreeTTS** to convert text into speech. The app provides an easy-to-use **GUI**, allowing users to select different voices, adjust speech rate and volume, and listen to synthesized speech in real-time.

## 🚀 Features
✔️ **Graphical User Interface** powered by JavaFX
✔️ **Text-to-Speech conversion** using FreeTTS
✔️ **Voice selection** (Kevin and Kevin16 (more to be added))
✔️ **Adjustable speech rate and volume**
✔️ **Modern UI styling** with CSS
✔️ **Maven-based project for easy setup**

## 🛠️ Technologies Used
- **Java 21** (JDK 21 recommended)
- **JavaFX 21** (Controls & FXML)
- **Maven** for dependency management
- **CSS** for UI styling
- **FreeTTS 1.2.2** for speech synthesis

## 📥 Installation & Setup
### Prerequisites
- **Java 21** installed (JDK 21 recommended)
- **Maven** installed

### Clone the repository
```sh
git clone https://github.com/asd1495/text2speech.git
cd text2speech

## 🖥️ Usage

-    Open the application.
-    Enter the text you want to convert to speech.
-    Select a voice (Kevin or Kevin16 are the one's available right now).
-    Adjust the rate and volume as needed.
-    Click "Speak" to hear the synthesized voice.
